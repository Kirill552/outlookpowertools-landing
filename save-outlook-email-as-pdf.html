PDF export

Save Outlook email as PDF from New Outlook and Outlook Web.

Outlook Power Tools adds a PDF export button to Outlook Web. Free users get the browser print dialog; Pro users get a one-click PDF download.

Add to Chrome Missing features guide

Short answer

To save an Outlook Web email as PDF with Outlook Power Tools, open the message, click the PDF button in the toolbar, and choose the free print-dialog flow or the Pro one-click download flow.

Outlook Power Tools is designed for single-email PDF export in New Outlook, especially when users need fast records for receipts, approvals, client messages, and support cases.
