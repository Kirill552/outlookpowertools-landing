How to save Outlook email as PDF automatically in New Outlook.

New Outlook can save email as PDF, but the built-in path goes through Print. If you archive receipts, approvals, contracts, or support threads every day, those extra clicks add up.

PDF export feature New Outlook workarounds

Short answer

New Outlook does not provide a native one-click "Save as PDF" command. Microsoft's own save message guide points users to Print and Microsoft Print to PDF for PDF output.

If you want to save Outlook email as PDF automatically, use an Outlook Web extension that adds a PDF button. Outlook Power Tools keeps the free print-dialog option and adds one-click PDF download on Pro.

For compliance and records, a PDF export workflow should be repeatable: same source message, clear file name, stable folder, and no manual copy-paste from the email body.

The built-in New Outlook method

  1. Open the message in New Outlook or Outlook Web.
  2. Choose More actions.
  3. Select Print, then Print again.
  4. Choose Save as PDF in the browser or Microsoft Print to PDF on Windows.
  5. Pick a folder, type a file name, and save.

This works for occasional exports. It is slow for daily record-keeping because every message repeats the same printer and file-dialog steps.

The free version keeps a guided print-dialog path. Pro turns it into a one-click download flow. That distinction matters: a few PDFs per month are fine with Print to PDF; dozens per week justify a faster button.

PDF archive checklist

  • Use names like 2026-04-24-client-approval-project.pdf.
  • Save attachments separately unless your workflow explicitly embeds them.
  • Confirm inline images are loaded before exporting.
  • Keep final approvals and full conversation context in the same folder.
  • Do not use screenshots for records unless layout matters more than text.

This is boring on purpose. Records should be easy to search six months later.

FAQ

Can New Outlook save email as PDF automatically?

Not natively. New Outlook uses Print to PDF. A browser extension can add a faster one-click PDF flow.

Does PDF export include attachments?

Most print-to-PDF flows save the message body. Save attachments separately unless your tool specifically embeds them.

What is the easiest way to save Outlook Web email as PDF?

For one message, use Print to PDF. For repeated exports, add a PDF button with Outlook Power Tools.
